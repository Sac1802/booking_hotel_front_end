# HotelFrontend

## Descripción
Aplicación web de reservas hoteleras desarrollada con Angular 21. Permite buscar hoteles, visualizar disponibilidad de habitaciones y gestionar reservas desde un panel de usuario.

## Objetivos
- Centralizar la búsqueda y filtrado de hoteles y habitaciones.
- Facilitar el flujo de reserva y pre‑reserva.
- Permitir a los usuarios autenticados ver y cancelar sus reservas.
- Mantener una arquitectura modular, escalable y con manejo de estado reactivo.

## Requisitos
- Node.js (recomendado LTS)
- npm (en `package.json` se usa `npm@11.9.0`)
- Angular CLI `^21.0.5` (puedes usar `npx ng` si no está global)
- Backend API disponible en `http://localhost:3000`

## Instalación
1. `npm install`

## Ejecución
1. `npm start`
2. Abrir `http://localhost:4200`

## Scripts útiles
- `npm run build`
- `npm run test`
